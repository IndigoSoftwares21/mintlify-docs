---
title: "List Product Variants"
api: "GET https://api.getscale.ng/api/v1/hub/shop/{shopId}/product/{productId}/variants"
---

Lists all configured variants, prices, options, and stock levels for a specific product.



### Authorizations

<ParamField header="X-Shop-API-Key" type="string" required>
  Your Secret API Key (`sk_live_...` or `sk_test_...`) generated from the Developer dashboard. Must be kept secret.
</ParamField>


### Path Parameters

<ParamField path="shopId" type="string" required>
  The unique UUID of the shop.
</ParamField>

<ParamField path="productId" type="number" required>
  The unique ID of the product.
</ParamField>


<ResponseExample>

```json 200 OK
{
  "status": "success",
  "data": [
    {
      "variantId": 12,
      "variantPrice": 1250000,

        {

        }
      ]
    }
  ]
}
```
