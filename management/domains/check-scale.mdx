---
title: "Check Subdomain Availability"
api: "GET https://api.sabivendor.com/api/v1/hub/shop/{shopId}/domains/check-scale/{subdomain}"
---

Verifies whether a desired Sabi Vendor subdomain is currently free for registration.



### Authorizations

<ParamField header="X-Shop-API-Key" type="string" required>
  Your Secret API Key (`sk_live_...` or `sk_test_...`) generated from the Developer dashboard. Must be kept secret.
</ParamField>


### Path Parameters

<ParamField path="shopId" type="string" required>
  The unique UUID of the shop.
</ParamField>

<ParamField path="subdomain" type="string" required>
  The subdomain name prefix to check (e.g. `myshop`).
</ParamField>


<ResponseExample>

```json 200 OK
{
  "status": "success",
  "data": {
    "available": true
  }
}
```

```json 400 Bad Request
{
  "message": "FORM_VALIDATION_ERROR",
  "code": 400,
  "validationErrors": [
    {
      "path": [
        "subdomain"
      ],
      "message": "Subdomain suffix matches invalid format"
    }
  ]
}
```

</ResponseExample>
