---
title: "Update Domain Settings"
api: "PATCH https://api.getscale.ng/api/v1/hub/shop/{shopId}/domains/{domainId}"
---

Modifies settings for custom domains, such as marking a domain as the primary checkout host or toggling active routing states.



### Authorizations

<ParamField header="X-Shop-API-Key" type="string" required>
  Your Secret API Key (`sk_live_...` or `sk_test_...`) generated from the Developer dashboard. Must be kept secret.
</ParamField>


### Path Parameters

<ParamField path="shopId" type="string" required>
  The unique UUID of the shop.
</ParamField>

<ParamField path="domainId" type="number" required>
  The unique ID of the domain record.
</ParamField>

### Request Body (JSON)

<ParamField body="isActive" type="boolean">
  Toggle whether traffic is routed to this domain.
</ParamField>

<ParamField body="isPrimary" type="boolean">
  Set as the primary root address of the storefront.
</ParamField>

{
  "status": "success",
  "message": "Domain settings updated successfully"
}

</ResponseExample>
