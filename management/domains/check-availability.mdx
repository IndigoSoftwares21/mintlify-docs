---
title: "Check Domain Availability"
api: "GET https://api.sabivendor.com/api/v1/hub/shop/{shopId}/domains/check-availability"
---

Checks availability details of an external custom domain in the project system database.



### Authorizations

<ParamField header="X-Shop-API-Key" type="string" required>
  Your Secret API Key (`sk_live_...` or `sk_test_...`) generated from the Developer dashboard. Must be kept secret.
</ParamField>


### Path Parameters

<ParamField path="shopId" type="string" required>
  The unique UUID of the shop.
</ParamField>

### Query Parameters

<ParamField query="domain" type="string" required>
  The custom domain address to check.
</ParamField>


<ResponseExample>

```json 200 OK
{
  "status": "success",
  "data": {
    "available": true
  }
}
```

```json 400 Bad Request
{
  "message": "FORM_VALIDATION_ERROR",
  "code": 400,
  "validationErrors": [
    {
      "path": [
        "domain"
      ],
      "message": "Invalid domain format"
    }
  ]
}
```

</ResponseExample>
