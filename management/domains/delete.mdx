---
title: "Delete Domain"
api: "DELETE https://api.getscale.ng/api/v1/hub/shop/{shopId}/domains/{domainId}"
---

Removes a registered custom domain or subdomain configuration.



### Authorizations

<ParamField header="X-Shop-API-Key" type="string" required>
  Your Secret API Key (`sk_live_...` or `sk_test_...`) generated from the Developer dashboard. Must be kept secret.
</ParamField>


### Path Parameters

<ParamField path="shopId" type="string" required>
  The unique UUID of the shop.
</ParamField>

<ParamField path="domainId" type="number" required>
  The unique ID of the domain record.
</ParamField>


<ResponseExample>

```json 200 OK
{
  "status": "success",
  "message": "Domain removed successfully"
}
```

```json 400 Bad Request
{
  "message": "FORM_VALIDATION_ERROR",
  "code": 400,
  "validationErrors": [
    {
      "path": [
        "domainId"
      ],
      "message": "Domain not found or does not belong to this shop"
    }
  ]
}
```

</ResponseExample>
