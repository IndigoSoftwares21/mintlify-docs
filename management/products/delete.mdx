---
title: "Delete Product"
api: "DELETE https://api.sabivendor.com/api/v1/hub/shop/{shopId}/product/{productId}"
---

Soft-deletes or archives a product from your inventory catalog. Once archived, the product will no longer appear on storefront index APIs.



### Authorizations

<ParamField header="X-Shop-API-Key" type="string" required>
  Your Secret API Key (`sk_live_...` or `sk_test_...`) generated from the Developer dashboard. Must be kept secret.
</ParamField>


### Path Parameters

<ParamField path="shopId" type="string" required>
  The unique UUID of the shop.
</ParamField>

<ParamField path="productId" type="number" required>
  The unique ID of the product.
</ParamField>


<ResponseExample>

```json 200 OK
{
  "status": "success",
  "message": "Product deleted successfully"
}
```

```json 400 Bad Request
{
  "message": "FORM_VALIDATION_ERROR",
  "code": 400,
  "validationErrors": [
    {
      "path": [
        "productId"
      ],
      "message": "Product not found"
    }
  ]
}
```

</ResponseExample>
