---
title: "List Products"
api: "GET https://api.getscale.ng/api/v1/hub/shop/{shopId}/product"
---

Retrieves a list of all products in your shop catalog. Use this endpoint for backend inventory tables, synchronization dashboards, and inventory status tracking.



### Authorizations

<ParamField header="X-Shop-API-Key" type="string" required>
  Your Secret API Key (`sk_live_...` or `sk_test_...`) generated from the Developer dashboard. Must be kept secret.
</ParamField>


### Path Parameters

<ParamField path="shopId" type="string" required>
  The unique UUID of the shop.
</ParamField>


<ResponseExample>

```json 200 OK
{
  "status": "success",
  "data": [
    {
      "id": 101,
      "productName": "Premium Running Shoes",
      "productPrice": 1500000,
      "stockQuantity": 100,
      "isActive": true
    }
  ]
}
```

```json 400 Bad Request
{
  "message": "FORM_VALIDATION_ERROR",
  "code": 400,
  "validationErrors": [
    {
      "path": [
        "shopId"
      ],
      "message": "Shop not found"
    }
  ]
}
```

</ResponseExample>
