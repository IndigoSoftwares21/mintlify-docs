---
title: "List Collections"
api: "GET https://api.getscale.ng/api/v1/hub/shop/{shopId}/collections"
---

Retrieves all collections configured for the shop.



### Authorizations

<ParamField header="X-Shop-API-Key" type="string" required>
  Your Secret API Key (`sk_live_...` or `sk_test_...`) generated from the Developer dashboard. Must be kept secret.
</ParamField>


### Path Parameters

<ParamField path="shopId" type="string" required>
  The unique UUID of the shop.
</ParamField>


<ResponseExample>

```json 200 OK
{
  "status": "success",
  "data": [
    {
      "id": 1,
      "name": "Winter Specials",
      "slug": "winter-specials",
      "imageUrl": null,
      "isActive": true
    }
  ]
}
```

```json 400 Bad Request
{
  "message": "FORM_VALIDATION_ERROR",
  "code": 400,
  "validationErrors": [
    {
      "path": [
        "shopId"
      ],
      "message": "Invalid uuid"
    }
  ]
}
```

</ResponseExample>
