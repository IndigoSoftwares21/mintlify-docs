---
title: "Get Products in Collection"
api: "GET https://api.getscale.ng/api/v1/hub/shop/{shopId}/collections/{collectionId}/products"
---

Retrieves a list of all products belonging to a collection.



### Authorizations

<ParamField header="X-Shop-API-Key" type="string" required>
  Your Secret API Key (`sk_live_...` or `sk_test_...`) generated from the Developer dashboard. Must be kept secret.
</ParamField>


### Path Parameters

<ParamField path="shopId" type="string" required>
  The unique UUID of the shop.
</ParamField>

<ParamField path="collectionId" type="number" required>
  The unique collection ID.
</ParamField>


<ResponseExample>

```json 200 OK
{
  "status": "success",
  "data": [
    {
      "id": 101,
      "productName": "Air Max Sneaker",
      "productPrice": 1200000,
      "isActive": true
    }
  ]
}
```

```json 400 Bad Request
{
  "message": "FORM_VALIDATION_ERROR",
  "code": 400,
  "validationErrors": [
    {
      "path": [
        "collectionId"
      ],
      "message": "Collection not found"
    }
  ]
}
```

</ResponseExample>
