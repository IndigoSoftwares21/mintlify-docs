---
title: "Update Storefront Settings"
api: "PUT https://api.sabivendor.com/api/v1/hub/shop/{shopId}/storefront-settings"
---

Update theme color, template IDs, font face selections, announcement bar parameters, and checkout options. Handles uploading a hero image file.



### Authorizations

<ParamField header="X-Shop-API-Key" type="string" required>
  Your Secret API Key (`sk_live_...` or `sk_test_...`) generated from the Developer dashboard. Must be kept secret.
</ParamField>


### Path Parameters

<ParamField path="shopId" type="string" required>
  The unique UUID of the shop.
</ParamField>

### Request Body (Multipart / Form Data)

<ParamField body="themeColor" type="string">
  Hex color code.
</ParamField>

<ParamField body="announcementBarText" type="string">
  Banner announcement message text.
</ParamField>

<ParamField body="showAnnouncementBar" type="boolean">
  Flag to display the announcement bar.
</ParamField>


<ResponseExample>

```json 200 OK
{
  "status": "success",
  "message": "Settings updated successfully",
  "data": {
    "themeColor": "#22162B",
    "announcementBarText": "Summer Mega Sale has started!",
    "showAnnouncementBar": true
  }
}
```

```json 400 Bad Request
{
  "message": "FORM_VALIDATION_ERROR",

      "path": [
```

</ResponseExample>
